# 关键词

| ------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| pass   | 当语法上需要一个语句，但程序需要什么动作也不做时，可以使用它 | 暂时不编写改部分的代码可以用其来占位，如果不写python解释器可能报错。 |
